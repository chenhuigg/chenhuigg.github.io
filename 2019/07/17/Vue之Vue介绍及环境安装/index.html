Vue是一套用于构建用户界面的“渐进式框架”，与其他大型框架不同的是，Vue被设计为可以自底向上逐层应用。Vue的核心库只关注视图层，不仅易于上手，还便于第三方库或既有项目整合。

前端框架三巨头：Vue.js、React.js、AngularJS，Vue.js以其轻量易用著称，Vue.js和React.js发展速度最快，AngularJS还是老大

首先按照nrm，这里 -g 代表全局安装，可能需要一点时间。
